# 🧸 GummyBear - Iron Performance

## 📌 المشروع

**GummyBear** هو مشروع مكملات غذائية ليبى على شكل حلوى هلامية Gummies، يتم استيراده من تونس وإعادة تسويقه محلياً عبر الصيدليات.

يتكون من 5 منتجات: Hair, Skin & Nails، No Stress، Multivitamin Sugar-Free، Sleep، Magnesium.

---

## 👥 الشراكة

- **شريك رأس المال**: يمتلك الشركة ويمول الاستيراد.
- **أنت**: طبيب غير ممارس، مسؤول عن التفاوض، التسويق، البيع، التحصيل.
- **مندوب مبيعات**: واحد.

## 🛠️ وظائف الموقع

### بطاقات المنتجات
- اسم المنتج مع إيموجي مميز
- 60 Gummies
- سعر المستهلك، سعر الصيدلية (قطعة)، سعر الصيدلية (25 قطعة + ستاند)
- هامش الربح وتاريخ الصلاحية
- زر "📚 اضغط للتفاصيل العلمية" يفتح نافذة منبثقة بالمعلومات الكاملة

### نافذة الطلبية (إنشاء طلبية)
- اسم الصيدلية (نصي)
- عنوان الصيدلية (نصي)
- تاريخ الفاتورة (تاريخ)
- رقم الفاتورة (اختياري – تلقائي يبدأ بـ 26)
- عرض الستاند (رقمي، عند تفعيله تُعطل حقول المنتجات وتُملأ تلقائياً بـ 5 × عدد العروض)
- كميات المنتجات الخمسة
- إجمالي الطلبية (يُحتسب تلقائياً)
- زر "📱 إرسال الطلب عبر واتساب"
- زر "🧾 إصدار فاتورة رسمية (PDF)"

### الفاتورة النهائية (PDF)
- ترويسة الشركة
- تاريخ الفاتورة، رقم الفاتورة، اسم الصيدلية وعنوانها
- جدول المنتجات (المنتج، سعر المستهلك، سعر القطعة، سعر العرض، الكمية، الإجمالي)
- العرض يشمل (ستاند مجاني)
- شروط الدفع (دفعات أسبوعية، شهر كحد أقصى، نقداً أو تحويل بنكي)
- الحساب البنكي لشريك رأس المال (اسم المستفيد، الفرع، رقم الحساب، IBAN)
- أرقام التواصل (واتساب، اتصال)
- توقيع المستلم واسمه
- تذييل

### المرجع العلمي (`science.html`)
- فهرس محتويات تفاعلي
- لكل منتج: معلومات عامة، وصف، التركيبة النشطة (جداول المكونات، الجرعات، الفوائد، المصادر العلمية)، التحذيرات، الفئات العمرية، التفاعلات الدوائية، الآثار الجانبية
- زر عودة إلى القائمة الرئيسية مع جملة "🔽 اضغط للرجوع"
- زر طائر "عودة للأعلى"

### الميزات التقنية
- الوضع الليلي (Dark Mode) تلقائي حسب إعدادات الجهاز
- تذييل موحد يحتوي على أيقونات واتساب وإنستغرام
- دعم كامل للطباعة (CSS `@media print`)
